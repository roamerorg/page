Features

  • 📚 Access Multiple Versions of Books: Explore different editions and formats of your favorite books.

  • 👥 Community Building: Join discussions, share reviews, and connect with fellow readers.

Tech Stack

  • 📝 Programming Languages: Golang for backend development and Solidity for smart contracts.

  • ⚙️ Frameworks: Cobra for building the CLI interface.

  • 💾 Database: PostgreSQL for robust and scalable data storage.

  • ☁️ Cloud Services: AWS for hosting and cloud infrastructure.

  • 🔒 Blockchain: Ethereum for deploying smart contracts.

  • 🌐 APIs: REST and GraphQL for seamless data communication.

  • 📚 Libraries: Viper for configuration management and GORM for ORM in Golang.
